This is a very basic and simple tool with which you can create a fairly strong password for websites, online games, phones. Please note: if you need a password for use in important services, we recommend that you use a professional desktop software instead of an online generator. This is not a professional tool. Use this generator to create common random passwords with a few mouse clicks.

This online password generator uses reliable crypto algorithms:

random_bytes

Generates an arbitrary length string of cryptographic random bytes that are suitable for cryptographic use, such as when generating salts, keys or initialization vectors.

https://php.net/manual/en/function.random-bytes.php
random_int

Generates cryptographic random integers that are suitable for use where unbiased results are critical...

https://php.net/manual/en/function.random-int.php

These algorithms are used together with nickfinder's own generator module.

Nickfinder.com does not store the generated passwords, so we can not restore them if you create a new set or leave the page.
Write down the password in a safe place before using it.
